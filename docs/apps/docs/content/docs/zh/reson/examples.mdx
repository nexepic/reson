## 指定语言

```bash
./reson \
  --source-path src \
  --languages rust
```

多语言：

```bash
./reson \
  --source-path src \
  --languages rust,python
```

## 排除目录与文件

```bash
./reson \
  --source-path src \
  --excludes "**/test/**"
```

```bash
./reson \
  --source-path src \
  --excludes "test,build"
```

```bash
./reson \
  --source-path src \
  --excludes "**/*.md"
```

```bash
./reson \
  --source-path src \
  --excludes "**/README.md"
```

## 设置阈值

```bash
./reson \
  --source-path src \
  --threshold 10
```

## 设置最小 AST 节点数

```bash
./reson \
  --source-path src \
  --threshold 10 \
  --min-ast-nodes 12
```

## 输出到文件

```bash
./reson \
  --source-path src \
  --output-format json \
  --output-file result.json
```
